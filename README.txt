

MIMO Wrapper Class -Dotnet
=====================================================

An implementation of the Mimo REST API using OAuth 2.0 in C#. 

Setup Project in Visual Studio

----------------------------


1.   Open the `MimoAPI.sln` with Visual Studio.

2.   Change the "ReturnURL" as per your domain something like "http://your domain here/Default.aspx".
3.   Click on `Build > Rebuild Solution` and click `View in Browser`.
